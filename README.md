- B类题

  ### 题目描述

  【编程题】

  给定一个整数数组，请实现就地重排，使得所有负整数在前，非负整数在后且各自保持原数组相对次序。

  ### 示例

  >输入：[2, -1, 4, -3, 0, -2, 3]
  >
  >输出：[-1, -3, -2, 2, 0, 4, 3]
  >
  >解释：其中-1，-3， -2 和 2， 4， 0， 3 均保持原数组次序
